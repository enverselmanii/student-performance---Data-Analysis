# Student Performance Analysis with Pandas

This project demonstrates practical data analysis skills using Python and Pandas on a realistic student performance dataset.

---

##  Project Overview
The goal of this project is to showcase:
- Data cleaning and preparation
- Feature engineering (Average score, Passed flag)
- Filtering & boolean masking
- GroupBy aggregations
- Data visualization for insights
- Interview-ready Jupyter Notebook

---

##  Dataset
- Columns: Student, Gender, Math, Python, Statistics, Hours_Studied
- Includes 6 example students
- Can be extended with CSV or other datasets

---

##  Features
- Calculate average scores and pass/fail status
- Identify students who did not pass
- Filter students based on study hours or performance
- Group and summarize grades by gender
- Clean and sort data for analysis
- Visualize average performance by gender
- Advanced filtering (bonus examples)

---

##  Visualization
- Bar chart showing **average score by gender**  
- Provides intuitive insight for comparison

---

##  Technologies
- Python 3
- Pandas
- Matplotlib
- Jupyter Notebook
